## 작성 방식

- 각 폴더 안에 주제별 `.md` 파일로 정리
- 이론 → 구현 → 성능 비교 → 깨달은 점 순서로 작성
- 이론 정리 커밋과 구현 커밋은 분리 (`study:` vs `feat:`)
